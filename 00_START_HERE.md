# ElectroBridge — Quick Reference Card

## Platform Overview
**Name:** ElectroBridge  
**Purpose:** Electronics & Semiconductor job/opportunity aggregator for Indian researchers  
**Stack:** Next.js + Supabase + Vercel (100% FREE)
